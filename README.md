# Алексеев Артём ИТС Лабораторная №4

## Задача 1

### Текст задачи

Дерево содержит строки. Добавить к каждой строке заданный символ в конец.
### Алгоритм решения
  1) Запрашиваем у пользователя данные исходного дерева с помощью функции readFromConsole и записываем в tree
  2) запрашиваем символ для добавки с помощью readChar
  3) Создаем новое дерево с помощью map
  4) Выводим исходное и новое деревья с помощью printTree

## Задача 2

### Текст задачи
Сформировать список из четных элементов дерева.
### Алгоритм решения
  1) Запрашиваем у пользователя данные исходного дерева с помощью функции readFromConsole и записываем в tree
  2) Выводим исходное дерево
  3) С помощью функции collectEvenNumbers получаем все четные числа дерева и записываем в evenNumbers
  4) Выводим четные числа
